# 0002. 공개 배포 안전 게이트

- 상태: accepted
- 날짜: 2026-07-14

## 배경

원본 저장소에는 이력서 원자료, 로컬 절대경로, 내부 검토 문서처럼 공개 사이트에 포함하면 안 되는 정보가 함께 존재합니다.

## 결정

- 새 저장소 안의 승인된 콘텐츠와 자산만 빌드 입력으로 사용합니다.
- 전화번호, ATS PDF, 원본 일감, 로컬 절대경로를 공개하지 않습니다.
- 공개 안전 검사와 링크 검사를 빌드 뒤 실행합니다.
- `PUBLIC_RELEASE`와 `PUBLIC_SITE_URL`이 모두 설정된 빌드만 검색 허용과 canonical URL을 출력합니다.

## 결과

Preview와 로컬 빌드는 기본적으로 검색 제외 상태이며, 공개 준비가 끝난 빌드만 검색 노출 상태로 전환됩니다.
